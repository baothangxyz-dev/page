# Краткое описание
Скрипт поднимающий web-server на localhost. Отображает текущий статус механической руки. Статус можно менять и он сохраняется в отдельный 

# Управление
1. Установите Python (если у вас его ещё нет)
2. Скачайте код или клонируйте репозиторий
3. Откройте терминал в директории, содержащей файл server.py
4. Выполните команду: `python server.py`
5. Откройте браузер: http://localhost:5000

# Примечание
В репозитории так же присутствует эскзиз робо-руки для которой был выполнен сервер.
