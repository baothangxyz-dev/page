# Краткое описание
Скрипт поднимающий web-server на localhost. Отображает текущий статус механической руки. Статус можно менять и он сохраняется в отдельный 

# Управление
Посмотреть текущий статус: http://baothangxyz-dev.github.io/page
Уставновить статус в ON: https://baothangxyz-dev.github.io/page/index.html?status=on
Установить статус в OFF: https://baothangxyz-dev.github.io/page/index.html?status=off

# Примечание
В репозитории так же присутствует эскзиз робо-руки для которой был выполнен сервер.
