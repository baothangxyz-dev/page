# Структурная схема проекта роборуки на базе CD4000
# Схема организации системы
Веб-интерфейс (index.html) → Серверная обработка (index.php) → Микроконтроллер Arduino → Блок управления на CD4000 → Драйвер двигателя → Двигатель постоянного тока → Роботизированная кисть.
# Применяемый двигатель
* Используемый двигатель: коллекторный двигатель постоянного тока с редуктором.
* Тип энкодера: инкрементальный магнитный энкодер, установленный на валу двигателя.
# Механизм остановки
* По положению (DIP-переключатели + энкодер)
* По касанию (микрик на пальце)
* По натяжению троса (микрик на пружине)
